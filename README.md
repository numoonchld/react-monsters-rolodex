# react-monsters-rolodex

`cd monsters-rolodex/`
    - go to project root dir 

`npm install` 
    - install dependencies in `package.json`

`npm run build` 
    - build source code to static files

`npm start`
    - serve source code via development server


- make project servable on github pages
